# Exercício 5 – Reflexão

## Resposta

Isso deixa claro para nós, programadores, que quem fez algo de errado fomos nós mesmos, seja uma vírgula esquecida, uma aspas não fechada ou qualquer outro detalhe no código. O computador executa exatamente as instruções que recebe, sem interpretar intenções ou corrigir erros. Por isso, é responsabilidade do programador escrever comandos corretos e revisar o código para garantir que o resultado seja o esperado.
